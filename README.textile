h1. ClockCard

p. ClockCard is a simple time tracking system currently being written in Java SE. ClockCard is being programmed as a semestral project for course PV168 Seminar in Java Programming taught in spring 2011 at Faculty of Infortmatics, Masaryk's University in Brno.
